# Antigravity Workspace Metadata

This folder is used by Antigravity (your AI coding assistant) to persist context and history for the **Edify Site** project.

## Folder Structure

- **conversations_summary.md**: A high-level overview of all recent sessions, their objectives, and outcomes.
- **history/**: Detailed logs of individual conversation sessions.
    - `current_session.md`: Tracks the ongoing dialogue.
    - `[id].md`: Archived logs of past sessions.

## Why keep this?
Keeping chat history in the workspace allows for better continuity across different sessions and helps track architectural decisions and changes over time.
